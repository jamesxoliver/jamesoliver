# Book

**Book:** A portable external memory artifact that preserves an organized pattern of thought so it can be reconstructed in other minds across space and time.

An author translates a structure inside their mind into a fixed sequence of symbols. A reader encounters those symbols and reconstructs the structure inside another mind. The reconstruction is never perfectly lossless, but the record prevents the thought from having to be remembered and translated again by every person through whom it passes.

The printed book is unusually durable because it combines several properties without maximizing any one of them. Each copy is stable enough to survive neglect, portable enough to travel, directly legible without a powered intermediary, and reproducible without altering the original. Stone survives because one instance resists destruction, while a transient signal survives only if it is continually reproduced or recorded. A printed book does both well enough: each instance can endure, and additional instances can be made.

Marcus Aurelius has been dead for nearly two thousand years. When someone reads *Meditations*, a pattern that once existed inside his mind is assembled again inside theirs.

A book allows thought to outlive the mind that produced it.
