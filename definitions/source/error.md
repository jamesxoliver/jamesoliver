# Error

**Error:** Any difference between what a representation asserts and reality within the conditions and resolution of the claim.

Error exists only within the representation's claim. A transit map that omits trees contains no error because it does not claim to represent trees. If it places two stations in the wrong order, the representation differs from reality along a relationship it claims to preserve, and that difference is error.

Compression and error are therefore not identical. Every representation excludes information, but exclusion becomes error only when the missing or distorted information belongs inside the claim's boundary.

Where the difference can be measured, error has magnitude. Where no comparison rule exists, it remains a discrepancy rather than a distance. Truth is binary: an assertive representation is true when it contains no error within the distinctions preserved by its claim. When the representation is a belief, error is the difference between what the mind takes to correspond to reality and reality itself.
