# Language

**Language:** A shared system of symbols and rules that encodes structure into transmissible patterns from which a compatible interpreter can reconstruct it.

Language is not inherently auditory. Its patterns can be heard, seen, or felt. Speech uses sound, signed languages use movement and space, writing uses visible or tactile marks, and each works only because independent interpreters share enough of the system to recover a bounded structure from the expression.

The expression does not contain everything the receiver reconstructs. The word *fire* contains no heat, color, movement, pain, light, smoke, or crackling. It carries a compressed pattern that causes a receiver to rebuild a model using memories and relationships already inside their mind. The sender transmits the archive, while the receiver supplies much of the decompressor.

Natural language is therefore permanently imperfect. Different minds can reconstruct different internal objects from the same expression because no two receivers contain exactly the same prior structure. Communication remains possible because shared rules and experience keep the reconstruction within a sufficiently similar region. A formal language can eliminate much of this variation by defining its symbols and transformations explicitly.

The test of a language is whether fixed expressions allow independent interpreters who understand the system to recover the same invariant structure with sufficient reliability. The sensory channel can change while this function remains.
