# Representation

**Representation:** A bounded form that stands for something, preserving selected structure while discarding the rest.

The word *fire* stands for fire, but it contains no heat, color, movement, sound, or capacity to burn. A map preserves selected relationships within a territory while leaving the territory itself absent. A representation works because some structure crosses the boundary even though the thing represented does not.

A representation may preserve everything inside a bounded description while remaining selective relative to the thing itself. A copied string can preserve every symbol, but the symbols still preserve only selected properties of what they represent. Selection and error are not identical. A transit map can omit trees and distort geographic distance without error when it claims to represent only station order. The boundary determines which differences matter.

If a form preserved every property and causal capacity of the thing, it would cease to stand in for the thing and become another instance of it.

Resemblance alone does not create representation. A cloud may resemble a face without standing for one. Representation begins when a form refers to something beyond itself.
